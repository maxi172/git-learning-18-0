  • git init - Команда которая инициализирует Гит-репозиторий
  • git config user email - это наша почта в которой производится момент сохранения
  • git config gui.config utf-8 поддержка кириллицы
  • git add - это команда добавляет в зону отслеживанмя для момента сохранения
  • git commit -m'сообщение сохранение' - момент сохранения
  • git status - это проверка в каком состаянии файлы находятся
  • gitk - это отрытия программы для просмотра репозитория
  • git branch - список веток. git branch имя cоздаться ветка с именем имя
  • git checkout - переходы по веткам и по комитам
  • git merge - слияние веток
  • git merge - conflict
